# Palindromo Com Deque

## Dificuldade
medio

## Descricao
Verifique se uma palavra e palindromo usando retiradas nas duas extremidades.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma palavra.

## Output
Retorna SIM ou NAO.

## Exemplos
Input:
radar

Output:
SIM

## Objetivo
Treinar comparacao de extremos.
