# Deque Monotonica Minima

## Dificuldade
dificil

## Descricao
Mantenha o menor elemento de uma janela deslizante usando deque monotonica.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o vetor e o tamanho da janela.

## Output
Retorna os minimos de cada janela.

## Exemplos
Input:
9 7 2 4 6 8
3

Output:
2 2 2 4

## Objetivo
Treinar remocao monotonica de candidatos.
