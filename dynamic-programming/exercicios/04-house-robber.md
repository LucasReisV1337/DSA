# House Robber

## Dificuldade
medio

## Descricao
Maximize a soma sem escolher casas adjacentes.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe os valores das casas.

## Output
Retorna o lucro maximo.

## Exemplos
Input:
2 7 9 3 1

Output:
12

## Objetivo
Treinar escolha otima com subestrutura.
