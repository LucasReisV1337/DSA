# Coin Change Minimo

## Dificuldade
medio

## Descricao
Encontre a menor quantidade de moedas para formar um valor alvo.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe as moedas e o valor alvo.

## Output
Retorna a quantidade minima ou -1.

## Exemplos
Input:
1 2 5
11

Output:
3

## Objetivo
Treinar DP de minimo.
