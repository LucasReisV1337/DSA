# Longest Common Subsequence

## Dificuldade
dificil

## Descricao
Calcule o tamanho da subsequencia comum mais longa entre duas strings.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe duas strings.

## Output
Retorna o tamanho da LCS.

## Exemplos
Input:
abcde
ace

Output:
3

## Objetivo
Treinar DP em duas dimensoes.
