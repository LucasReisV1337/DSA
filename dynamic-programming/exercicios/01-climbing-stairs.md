# Climbing Stairs

## Dificuldade
facil

## Descricao
Conte quantas formas existem para subir uma escada de n degraus usando passos de 1 ou 2.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe n.

## Output
Retorna a quantidade de formas.

## Exemplos
Input:
5

Output:
8

## Objetivo
Treinar recorrencia linear.
