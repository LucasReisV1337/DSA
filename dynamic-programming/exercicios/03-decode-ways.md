# Decode Ways

## Dificuldade
medio

## Descricao
Conte quantas decodificacoes validas existem para uma string numerica.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a string numerica.

## Output
Retorna a quantidade de decodificacoes.

## Exemplos
Input:
226

Output:
3

## Objetivo
Treinar DP em strings.
