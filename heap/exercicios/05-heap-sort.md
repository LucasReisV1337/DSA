# Heap Sort

## Dificuldade
dificil

## Descricao
Ordene um vetor usando heap sort completo.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe um vetor de inteiros.

## Output
Retorna o vetor em ordem crescente.

## Exemplos
Input:
4 10 3 5 1

Output:
1 3 4 5 10

## Objetivo
Treinar ordenacao baseada em heap.
