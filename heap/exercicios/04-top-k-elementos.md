# Top K Elementos

## Dificuldade
medio

## Descricao
Retorne os k maiores elementos de uma lista sem ordenar toda a lista.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a lista e k.

## Output
Retorna os k maiores em ordem decrescente.

## Exemplos
Input:
7 1 9 2 8 4
3

Output:
9 8 7

## Objetivo
Treinar selecao eficiente com heap.
