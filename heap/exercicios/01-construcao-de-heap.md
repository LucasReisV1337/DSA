# Construcao De Heap

## Dificuldade
medio

## Descricao
Transforme um vetor em um min-heap valido.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe um vetor de inteiros.

## Output
Retorna o vetor reorganizado como heap.

## Exemplos
Input:
9 4 7 1 0 3 2

Output:
0 1 2 4 9 3 7

## Objetivo
Treinar heapify de baixo para cima.
