# Escalonamento Round Robin

## Dificuldade
medio

## Descricao
Simule atendimento por fatias de tempo usando uma fila de processos.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe processos com tempo de execucao e o quantum.

## Output
Retorna a ordem de conclusao dos processos.

## Exemplos
Input:
P1 5
P2 3
P3 2
Q 2

Output:
P3 P2 P1

## Objetivo
Treinar processamento em FIFO com reentrada.
