# Traversal Em Largura

## Dificuldade
medio

## Descricao
Dada uma arvore binaria representada por adjacencia, produza a ordem de visita em largura.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o no raiz e a lista de filhos por nivel.

## Output
Retorna a ordem BFS.

## Exemplos
Input:
1:[2,3]
2:[4,5]
3:[6]

Output:
1 2 3 4 5 6

## Objetivo
Treinar uso de fila em BFS.
