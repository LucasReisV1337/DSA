# Fila Com Duas Pilhas

## Dificuldade
dificil

## Descricao
Implemente uma fila usando duas pilhas, mantendo FIFO.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe operacoes de enqueue e dequeue.

## Output
Retorna o estado da fila e os valores removidos.

## Exemplos
Input:
ENQ 1
ENQ 2
DEQ

Output:
1

## Objetivo
Treinar simulacao de fila sobre pilhas.
