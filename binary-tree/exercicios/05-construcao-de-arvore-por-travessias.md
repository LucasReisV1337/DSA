# Construcao De Arvore Por Travessias

## Dificuldade
dificil

## Descricao
Reconstrua uma arvore binaria a partir dos percursos inorder e postorder.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe as duas sequencias de visita.

## Output
Retorna o preorder da arvore reconstruida.

## Exemplos
Input:
4 2 5 1 3
4 5 2 3 1

Output:
1 2 4 5 3

## Objetivo
Treinar reconstrucao de arvore a partir de travessias.
