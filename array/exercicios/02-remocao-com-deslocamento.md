# Remocao Com Deslocamento

## Dificuldade
facil

## Descricao
Remova a primeira ocorrencia de um valor de um vetor e compacte o restante.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o vetor e o valor a remover.

## Output
Retorna o vetor final sem o valor removido.

## Exemplos
Input:
5 3 8 3 1
3

Output:
5 8 3 1

## Objetivo
Treinar exclusao com deslocamento linear.
