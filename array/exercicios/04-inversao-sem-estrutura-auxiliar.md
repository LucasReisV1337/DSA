# Inversao Sem Estrutura Auxiliar

## Dificuldade
dificil

## Descricao
Inverta um vetor no proprio espaco, sem criar outro vetor do mesmo tamanho.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o vetor de inteiros.

## Output
Retorna o vetor invertido.

## Exemplos
Input:
1 2 3 4 5

Output:
5 4 3 2 1

## Objetivo
Treinar manipulacao in-place.
