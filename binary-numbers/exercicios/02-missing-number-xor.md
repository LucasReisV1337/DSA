# Missing Number Com XOR

## Dificuldade
facil

## Descricao
Encontre o numero ausente em uma sequencia de 0 ate n.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a lista com um numero faltando.

## Output
Retorna o numero ausente.

## Exemplos
Input:
0 1 3

Output:
2

## Objetivo
Treinar propriedade de XOR para eliminacao.
