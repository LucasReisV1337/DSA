# Contagem De Bits

## Dificuldade
facil

## Descricao
Conte quantos bits 1 existem na representacao binaria de um inteiro nao negativo.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe um inteiro nao negativo.

## Output
Retorna a quantidade de bits 1.

## Exemplos
Input:
13

Output:
3

## Objetivo
Treinar manipulacao de bits e representacao binaria.
