# Palindromo Com Pilha

## Dificuldade
medio

## Descricao
Use uma pilha para comparar uma palavra com sua ordem inversa.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma palavra.

## Output
Retorna SIM ou NAO.

## Exemplos
Input:
arara

Output:
SIM

## Objetivo
Treinar reversao via LIFO.
