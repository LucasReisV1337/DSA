# Avaliacao Posfixa

## Dificuldade
medio

## Descricao
Avalie uma expressao matematica em notacao posfixa.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe operandos e operadores separados por espaco.

## Output
Retorna o resultado numerico.

## Exemplos
Input:
3 4 + 2 *

Output:
14

## Objetivo
Treinar processamento de expressao com pilha.
