# Validacao De Parenteses

## Dificuldade
facil

## Descricao
Verifique se uma string de parenteses, colchetes e chaves esta corretamente balanceada.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma string com simbolos de agrupamento.

## Output
Retorna SIM ou NAO.

## Exemplos
Input:
([{}])

Output:
SIM

## Objetivo
Treinar uso classico de pilha.
