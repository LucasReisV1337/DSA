# Min Stack

## Dificuldade
medio

## Descricao
Implemente uma pilha que retorna o menor elemento em O(1).

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe operacoes push, pop e min.

## Output
Retorna o estado da pilha e o menor valor apos cada consulta.

## Exemplos
Input:
push 5
push 2
min

Output:
2

## Objetivo
Treinar estrutura auxiliar em pilha.
