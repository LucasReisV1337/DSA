# Clone De Grafo

## Dificuldade
medio

## Descricao
Crie uma copia profunda de um grafo com vertices compartilhados.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a lista de adjacencia.

## Output
Retorna a estrutura clonada.

## Exemplos
Input:
1:2,3
2:1
3:1

Output:
1:2,3
2:1
3:1

## Objetivo
Treinar travessia e mapeamento de nos.
