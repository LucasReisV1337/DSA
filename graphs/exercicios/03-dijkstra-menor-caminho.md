# Dijkstra Menor Caminho

## Dificuldade
dificil

## Descricao
Calcule a menor distancia de uma origem para todos os vertices em grafo ponderado.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe arestas com pesos e a origem.

## Output
Retorna as distancias minimas.

## Exemplos
Input:
1-2:4
1-3:2
3-2:1
2-4:5
3-4:8
1

Output:
1:0
2:3
3:2
4:8

## Objetivo
Treinar caminho minimo com heap.
