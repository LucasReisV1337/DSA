# Network Delay Time

## Dificuldade
dificil

## Descricao
Dado um conjunto de rotas com tempo, determine o tempo para alcancar todos os nos.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe arestas, origem e quantidade de vertices.

## Output
Retorna o tempo total ou -1.

## Exemplos
Input:
1->2:1
1->3:4
2->3:2
3->4:1
1
4

Output:
4

## Objetivo
Treinar Dijkstra em rede.
