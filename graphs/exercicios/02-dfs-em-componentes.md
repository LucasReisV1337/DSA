# DFS Em Componentes

## Dificuldade
medio

## Descricao
Encontre todos os vertices visitados por DFS e o numero de componentes conectados.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a lista de adjacencia.

## Output
Retorna a ordem de visita e a quantidade de componentes.

## Exemplos
Input:
1:2
2:1
3:4
4:3

Output:
1 2 3 4
2

## Objetivo
Treinar busca em profundidade.
