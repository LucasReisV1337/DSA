# Remocao Em Lista Dupla

## Dificuldade
medio

## Descricao
Remova um no conhecido de uma lista duplamente encadeada atualizando ambos os ponteiros.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a lista e a chave a remover.

## Output
Retorna a lista final.

## Exemplos
Input:
1 2 3 4
3

Output:
1 2 4

## Objetivo
Treinar atualizacao de prev e next.
