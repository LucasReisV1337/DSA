# Uniao De Listas Ordenadas

## Dificuldade
dificil

## Descricao
Una duas listas encadeadas ordenadas em uma terceira lista ordenada.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe duas listas ordenadas.

## Output
Retorna a lista mesclada ordenada.

## Exemplos
Input:
1 3 5
2 4 6

Output:
1 2 3 4 5 6

## Objetivo
Treinar merge de listas ordenadas.
