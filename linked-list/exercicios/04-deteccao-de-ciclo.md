# Deteccao De Ciclo

## Dificuldade
medio

## Descricao
Determine se uma lista encadeada possui ciclo.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a sequencia de proximos nos nos.

## Output
Retorna SIM ou NAO.

## Exemplos
Input:
1->2->3->2

Output:
SIM

## Objetivo
Treinar tecnica de ponteiros rapidos e lentos.
