# Reversao De Lista

## Dificuldade
medio

## Descricao
Reverta uma lista simplesmente encadeada sem usar lista auxiliar.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe os valores da lista.

## Output
Retorna os valores na ordem inversa.

## Exemplos
Input:
1 2 3 4

Output:
4 3 2 1

## Objetivo
Treinar reversao de encadeamento.
