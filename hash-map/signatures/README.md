# Signatures

Esta subpasta reúne materiais sobre assinaturas, filtros e indexação baseada em codificação resumida de campos.

## Tópicos
- Assinaturas de registros
- Combinação de bits por campo
- Filtros de assinatura
- Bloom filters como técnica de pré-filtragem

## Uso típico
- Reduzir buscas desnecessárias
- Acelerar filtragem de registros por atributos
- Trabalhar com conjuntos aproximados de candidatos
