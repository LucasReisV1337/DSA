# Assinatura De Registros

## Dificuldade
medio

## Descricao
Gere uma assinatura binaria para cada conjunto de campos de um registro.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe os campos do registro.

## Output
Retorna a assinatura em binario.

## Exemplos
Input:
cargo dept salario

Output:
10110010

## Objetivo
Treinar codificacao resumida de atributos.
