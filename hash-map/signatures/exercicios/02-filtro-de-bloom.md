# Filtro De Bloom

## Dificuldade
dificil

## Descricao
Verifique se um elemento provavelmente pertence a um conjunto usando um filtro de Bloom simples.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o conjunto, o elemento e as funcoes de hash.

## Output
Retorna PODE_CONTER ou NAO_CONTER.

## Exemplos
Input:
ana
ana

Output:
PODE_CONTER

## Objetivo
Treinar filtragem probabilistica por assinatura.
