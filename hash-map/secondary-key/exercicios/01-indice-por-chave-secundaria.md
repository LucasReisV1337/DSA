# Indice Por Chave Secundaria

## Dificuldade
medio

## Descricao
Crie um indice que permita buscar registros por um campo secundario.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe registros com chave primaria e secundaria.

## Output
Retorna os registros associados a cada chave secundaria.

## Exemplos
Input:
1 Ana A
2 Bruno B
3 Alice A
A

Output:
1 Ana A
3 Alice A

## Objetivo
Treinar indexacao por atributo nao primario.
