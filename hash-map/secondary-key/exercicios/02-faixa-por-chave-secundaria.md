# Faixa Por Chave Secundaria

## Dificuldade
medio

## Descricao
Retorne todos os registros cuja chave secundaria esteja dentro de um intervalo.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe registros e um intervalo de chaves secundarias.

## Output
Retorna os registros filtrados.

## Exemplos
Input:
1 10
2 20
3 30
15 25

Output:
2 20

## Objetivo
Treinar filtragem com indice auxiliar.
