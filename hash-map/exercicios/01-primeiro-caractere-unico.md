# Primeiro Caractere Unico

## Dificuldade
facil

## Descricao
Dada uma string, encontre o primeiro caractere que nao se repete.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma string em uma linha.

## Output
Retorna o caractere ou -1.

## Exemplos
Input:
loveleetcode

Output:
v

## Objetivo
Treinar contagem de frequencia com mapa.
