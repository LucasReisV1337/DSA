# Agrupar Anagramas

## Dificuldade
medio

## Descricao
Agrupe palavras que sao anagramas entre si.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma lista de palavras.

## Output
Retorna os grupos separados.

## Exemplos
Input:
eat tea tan ate nat bat

Output:
eat tea ate
tan nat
bat

## Objetivo
Treinar normalizacao de chave.
