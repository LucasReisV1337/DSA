# Contagem De Frequencia

## Dificuldade
facil

## Descricao
Conte quantas vezes cada numero aparece em uma lista.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe a lista de inteiros.

## Output
Retorna pares numero e frequencia.

## Exemplos
Input:
1 2 2 3 3 3

Output:
1:1
2:2
3:3

## Objetivo
Treinar tabela de frequencia.
