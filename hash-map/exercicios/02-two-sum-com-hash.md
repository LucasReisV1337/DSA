# Two Sum Com Hash

## Dificuldade
facil

## Descricao
Encontre dois indices cujo valor some o alvo.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o vetor e o alvo.

## Output
Retorna os dois indices ou -1.

## Exemplos
Input:
2 7 11 15
9

Output:
0 1

## Objetivo
Treinar lookup O(1) com HashMap.
