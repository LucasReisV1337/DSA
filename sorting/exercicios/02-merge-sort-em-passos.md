# Merge Sort Em Passos

## Dificuldade
medio

## Descricao
Execute merge sort e mostre as fusoes intermediarias.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe um vetor de inteiros.

## Output
Retorna o vetor ordenado.

## Exemplos
Input:
8 3 7 4 9 2

Output:
2 3 4 7 8 9

## Objetivo
Treinar divisao e conquista.
