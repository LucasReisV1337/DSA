# Trace Bubble Sort

## Dificuldade
facil

## Descricao
Mostre o estado do vetor apos cada passada do bubble sort.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe um vetor de inteiros.

## Output
Retorna o vetor apos cada passada.

## Exemplos
Input:
5 1 4 2

Output:
1 4 2 5
1 2 4 5

## Objetivo
Treinar comparacao e troca em ordenacao simples.
