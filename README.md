## 📖 Convenções do Repositório / Repository Conventions

### Nomenclatura de Arquivos
- **Exercícios LeetCode**: `[NUMERO].[nome-do-exercicio].py`
  - Exemplo: `155.min-stack.py`, `020.valid-parentheses.py`
- **Implementações Base**: `[nome-da-estrutura].py`
  - Exemplo: `binary-tree.py`, `min-heap.py`
- **Técnicas/Padrões**: `[nome-da-tecnica].py`
  - Exemplo: `binary-search.py`, `sliding-window.py`

### Status dos Exercícios
| Símbolo | Significado | Meaning |
|---------|------------|---------|
| ✅ | Resolvido | Solved |
| ⏳ | Em progresso | In progress |
| ❌ | Bloqueado/Pendente | Blocked/Pending |

### Estrutura de um README de Tópico
Cada pasta temática deve incluir:
1. **O Que É?** — Explicação clara
2. **Quando Usar?** — Casos práticos
3. **Complexidade** — Tabela de operações
4. **Tabela de Exercícios** — Número, título, dificuldade, status, link
5. **Implementações** — Arquivo base

## 🚀 Começar / Getting Started

1. **Clone o repositório**
   ```bash
   git clone https://github.com/LucasReisV1337/DSA.git
   cd DSA
   ```

2. **Escolha um tópico** do [Índice](#-índice-de-estruturas--index)

3. **Leia o README** do tópico para entender a estrutura

4. **Revise os exercícios** na tabela do tópico

5. **Execute o código**
   ```bash
   python3 [pasta]/[numero].[exercicio].py
   ```
