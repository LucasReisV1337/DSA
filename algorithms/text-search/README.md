# Text Search

Esta pasta reúne materiais sobre busca de padrões em texto.

## Tópicos
- Busca de padrões com autômatos finitos
- Knuth-Morris-Pratt (KMP)
- Rabin-Karp
- Boyer-Moore
- Comparação de complexidades e pré-processamento

## Uso típico
- Encontrar uma palavra ou assinatura em um texto grande
- Comparar algoritmos exatos de correspondência de padrões
- Resolver exercícios de busca em strings
