# Tabela Prefixo KMP

## Dificuldade
medio

## Descricao
Monte a tabela de prefixo do algoritmo KMP para um padrao dado.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o padrao.

## Output
Retorna a tabela prefixo.

## Exemplos
Input:
ababaca

Output:
0 0 1 2 3 0 1

## Objetivo
Treinar pre-processamento do KMP.
