# Primeira Ocorrencia Rabin Karp

## Dificuldade
medio

## Descricao
Localize a primeira ocorrencia de um padrao em um texto usando hashing de janela.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe texto e padrao.

## Output
Retorna o indice da primeira ocorrencia ou -1.

## Exemplos
Input:
abracadabra
abra

Output:
0

## Objetivo
Treinar comparacao de hash e confirmacao final.
