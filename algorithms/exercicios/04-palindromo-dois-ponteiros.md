# Palindromo Com Dois Ponteiros

## Dificuldade
medio

## Descricao
Verifique se uma palavra e palindromo comparando caracteres dos extremos para o centro.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma palavra em uma linha.

## Output
Retorna se a palavra e palindromo ou nao.

## Exemplos
Input:
arara

Output:
SIM

## Objetivo
Treinar tecnica de dois ponteiros.
