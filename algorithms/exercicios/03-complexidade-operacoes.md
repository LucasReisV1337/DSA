# Complexidade De Operacoes

## Dificuldade
medio

## Descricao
Classifique uma lista de operacoes em O(1), O(log n) ou O(n) com base em sua descricao.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma operacao por linha.

## Output
Retorna a classe de complexidade para cada operacao.

## Exemplos
Input:
acesso indexado
busca binaria
percorrer lista

Output:
O(1)
O(log n)
O(n)

## Objetivo
Treinar leitura de custo assintotico.
