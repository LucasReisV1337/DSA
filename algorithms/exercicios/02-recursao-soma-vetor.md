# Recursao Para Soma De Vetor

## Dificuldade
facil

## Descricao
Calcule a soma de um vetor de inteiros usando recursao pura.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma lista de inteiros em uma linha.

## Output
Retorna a soma total.

## Exemplos
Input:
2 4 6 8

Output:
20

## Objetivo
Treinar decomposicao recursiva e caso base.
