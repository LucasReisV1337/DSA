# Construcao De Huffman

## Dificuldade
dificil

## Descricao
Construa uma arvore de Huffman a partir das frequencias dos simbolos.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe uma string de texto.

## Output
Retorna os codigos gerados para cada simbolo.

## Exemplos
Input:
banana

Output:
a:0
n:10
b:11

## Objetivo
Treinar compressao sem perdas por frequencia.
