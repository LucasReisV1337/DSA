# Taxa De Compressao

## Dificuldade
medio

## Descricao
Compare o tamanho original e o tamanho compactado de um texto usando codificacao dada.

## Regras de Negocio
- O algoritmo deve lidar corretamente com entrada vazia quando aplicavel.
- Nao use solucao pronta da biblioteca para o objetivo principal.
- Mantenha saida deterministica para o mesmo input.

## Input
Recebe o tamanho original e o tamanho compactado.

## Output
Retorna a taxa de compressao em percentual.

## Exemplos
Input:
1000
250

Output:
75%

## Objetivo
Treinar interpretacao de ganhos de compressao.
