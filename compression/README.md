# Compression

Esta pasta reúne materiais sobre compressão de dados e representação mais compacta de informação.

## Tópicos
- Ideia geral de compressão sem perdas
- Codificação por frequência
- Árvores de prefixo e codificação hierárquica
- Trade-offs entre taxa de compressão, tempo e memória

## Uso típico
- Reduzir espaço de armazenamento
- Acelerar transferência de dados
- Preparar blocos para leitura e escrita em arquivos grandes
